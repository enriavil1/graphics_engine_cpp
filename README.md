# 3D Graphics Engine written in C++

## Supported platforms

1. MacOS


## How to run
### Prerequisites 
- C++ 20
- CMAKE Version 3.9
- OpenGL

#### MacOS
1. sh {project_path}/build.sh
2. {project_path}/build/3d_engine_cpp
